# SwiftSkale Data Analytics Bootcamp — Cohort 7 (Lesson Hub)

An elegant, mobile-first educational presentation website for **SwiftSkale Data Analytics Bootcamp Cohort 7**.

It presents **Lesson 1 — Getting Started with Data Analytics & Business Intelligence** as a polished, presentation-style learning experience, with a navigation shell ready for **Lesson 2** and **Lesson 3**.

## What's inside

- `index.html` — the complete, self-contained website (HTML + CSS + JS in one file).
- `images/` — SwiftSkale logos and AI tool logos used across the site.
- `lesson-1/` — original training materials, linked as downloadable resources.

## Design

- **Headers:** Poppins · **Body:** Inter · **Code/data:** JetBrains Mono
- **Icons:** one consistent family — [Lucide](https://lucide.dev) (loaded via CDN, uniform stroke weight) used throughout.
- **Brand:** SwiftSkale magenta→purple gradient on a clean white base, with a dark gradient hero/footer.
- **Visual-first:** lesson concepts are presented as infographics, not walls of text — a data-type icon grid, a data-generation source flow, a circular data lifecycle (with mobile timeline), three-column DA/DS/BI comparison cards, a skills grid, a 7-step BI workflow stepper, and an AI-Powered Analytics section.
- Fully responsive (mobile, tablet, desktop), sticky navigation, scroll progress bar, animated section reveals, and an accessible accordion for activities.

## Deploy to GitHub Pages

1. Commit and push this folder to your repository's `main` branch.
2. On GitHub: **Settings → Pages**.
3. Under **Build and deployment**, set **Source = Deploy from a branch**, **Branch = `main`**, folder = `/ (root)`.
4. Save. Your site goes live at `https://<username>.github.io/<repo>/` within a minute or two.

Because `index.html` is at the repo root and links assets with relative paths, no build step is required.
